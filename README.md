LexiLearn is an intelligent reading-assistance web app designed to help **dyslexic learners** improve reading fluency through **voice-based practice**, **real-time feedback**, and **personalized progress tracking**.

Built with **React (Vite)** + **FastAPI** + **SQLModel**, LexiLearn provides students, parents, and teachers with role-based dashboards and learning analytics.


TO RUN BACKEND

run `cd backend`
run `venv\Scripts\activate`
run `pip install -r requirements.txt`
run `uvicorn app.main:app --reload`

TO RUN FRONTEND

run `cd frontend`
run `npm i`
run `npm run dev`
